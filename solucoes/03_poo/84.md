## Pergunta

84 - Escreve o código de um método genérico que instancie e devolva um _array_
de objetos do tipo genérico `T`. O tamanho do _array_ e o valor inicial de
todos os elementos do _array_ devem ser passados como argumentos opcionais
do método, cujos valores por omissão são 10 e `default(T)`, respetivamente.
Mostra 5 formas diferentes de usar o método (com tipos diferentes,
especificando ou não todos os parâmetros, indicando o nome dos parâmetros,
trocando a ordem dos mesmos, etc).

## Soluções

### Solução 1

```csharp
using System;

namespace Ex84_03poo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1 - Sem passar qualquer parâmetro.
            char[] myChars = CreateObjectArray<char>();

            // 2 - Especificando o parâmetro `value`.
            char[][] alotOfChars = CreateObjectArray<char[]>
                (value: new char[]{ 'a', 'b', 'c'});

            // 3 - Especificando todos os parâmetros
            char[][] moreOfMyChars = CreateObjectArray<char[]>(27, myChars);

            // 4 - Trocar a ordem dos parâmetros
            char[] str = CreateObjectArray<char>(value: '!', arraySize: 9);

            // 5 - Usando outros tipos
            Tuple<string[], string[]> tupleStrings;
            tupleStrings = new Tuple<string[], string[]>
                (CreateObjectArray<string>(value: "Hello"), 
                CreateObjectArray<string>(value: "World"));
        }

        public static T[] CreateObjectArray<T>(int arraySize = 10, T value = default(T))
        {
            T[] newGenericArray = new T[arraySize];

            for (int i = 0; i < newGenericArray.Length; i++)
            {
                newGenericArray[i] = value;
            }

            return newGenericArray;
        }
    }
}
```
