## Pergunta

47 - Indica o que é impresso no ecrã pelo seguinte código. Justifica a tua resposta,
	explicando detalhadamente em que consiste a operação efetuada pelo método Fact().

```cs
static void Main(string[] args)
{
    uint n = 5;
    Console.WriteLine($"{n}! = {Fact(n)}");
}

static uint Fact(uint i) => i > 1 ? i * Fact(i - 1) : 1;
```

## Soluções

### Solução 1

	O metodo Fact() retorna um uint i, que caso(if) este seja maior que um o metodo
	procede com a operação de i*Fact(i-1) e caso(else if) i não seja maior que 1
	então o metodo retorna 1.

	Assim sendo, é impresso o seguinte:
		5! = 5 * Fact(4)
